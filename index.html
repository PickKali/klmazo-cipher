Here's a really cursed set of ciphers.

How do these even work?

K Toggle: Biscuit Cipher
Gets the first letter and last letter, reverses the remaining until there no more letters.
The quick brown fox jumps over the lazy dog.
T.gheod q yuizackl behrot wnre fvoox s jpmu
It's like a cat making biscuits!
L Toggle: Adjacent Scrabble Cipher
This cipher takes each vowel and changes it to the sum of the scrabble point values of the adjacent characters. A = 0, E = 1, I = 2, O = 3, U = 4.
The quick brown fox jumps over the lazy dog.
Th5 q15ick br8wn f15x j15mps 7v6r th5 l11zy d7g.
What? No blank tiles?
M Toggle: ALICE THREE Cipher
There was a Alice in Borderland promotional poster that uses the card value to get the specific letter from a card suit. This isn't very good for longer messages.
